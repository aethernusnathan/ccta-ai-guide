## Why this exists

Cardiologists evaluating CCTA AI tools face a market flooded with vendor whitepapers and conference slides. Independent, rigorous, peer-reviewed comparisons are rare. This guide fills that gap.

It covers every commercially active FFR<sub>CT</sub> and AI-QCPA platform currently FDA-cleared, assessed entirely from public evidence.
